Cara Pakai Aplikasi

  • Pilih Bulan menggunakan tanda panah kiri/kanan.
  • Gunakan kotak Cari untuk menemukan nama bidan dengan cepat.
  • Klik pada kotak kosong di bawah tanggal untuk memilih Shift (P/S/M).
  • Klik pada Angka Tanggal di bagian atas untuk menandai Libur/Tanggal Merah.
  • Tekan tombol 💾 Simpan agar data tidak hilang (tersimpan ke server).
  • Gunakan ikon 📸 Kamera untuk menyimpan jadwal dalam bentuk foto/gambar.
  • Gunakan ikon 📥 Tanda Panah untuk mengunduh file Excel.
